10 базових команд роботи в системі Git
  • git init — Ініціалізує новий порожній Git-репозиторій у поточній директорії.
  • git clone [url] — Клонує існуючий репозиторій з віддаленого сервера на локальну машину.
  • git add [file] — Додає конкретний файл до індексації, готуючи його до коміту.
  • git add . — Додає всі зміни у файлах в поточній директорії до індексації.
  • git commit -m "message" — Фіксує зміни, додані в індексацію, з описовим повідомленням коміту.
  • git status — Відображає статус файлів у робочій директорії, показуючи, які файли змінені, додані або готові до коміту.
  • git log — Виводить історію комітів для поточної гілки.
  • git branch — Показує список всіх гілок у репозиторії або створює нову гілку.
  • git checkout [branch] — Перемикає робочу директорію на вказану гілку або коміт.
  • git push — Відправляє коміти з локальної гілки на віддалений репозиторій (наприклад, на GitHub).
